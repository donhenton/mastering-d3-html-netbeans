JQuery Image Timer

This form displays images and allows for setting various parameters for the length of the display. It is intended to be used in art instruction for short rapid gesture drawings.

The corresponding java code that handles these actions can be found in com.dhenton9000.spring.mvc.controllers.jquery MainJQueryController
